# Carlos Gomez — Portfolio

Personal portfolio and project showcase. Built with [SvelteKit](https://kit.svelte.dev/) 5, [Tailwind CSS](https://tailwindcss.com/) 4, and TypeScript.

## Stack

- **Framework:** Svelte 5
- **Styling:** Tailwind CSS 4
- **Tooling:** Vite 7, TypeScript, ESLint, Prettier

## Run locally

`npm install`
`npm run dev`
